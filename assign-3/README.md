# Assignment 3

This folder contains Python exercises demonstrating functions and math operations.

## Tasks

### Task 1: Factorial Calculator (`task1.py`)
A program that calculates the factorial of a given number using a function.

**Features:**
- User-defined function to calculate factorial
- Handles edge cases (0, 1, and negative numbers)
- Uses iterative approach with a for loop
- Input validation for negative numbers

**Usage:**
```bash
python task1.py
```

**Example:**
```
Enter a number to calculate its factorial: 5
The factorial of 5 is 120
```

### Task 2: Math Operations (`task2.py`)
A program that performs various mathematical operations using Python's math module.

**Features:**
- Calculates square root using `math.sqrt()`
- Computes natural logarithm using `math.log()`
- Calculates sine value using `math.sin()`
- Demonstrates Python's built-in math library

**Usage:**
```bash
python task2.py
```

**Example:**
```
Enter a number to calculate its square root: 16
Square root :4.0
Logarithem : 2.772588722239781
Sine : -0.2879033166650653
```
