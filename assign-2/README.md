# Assignment 2

This folder contains Python exercises demonstrating control flow and loops.

## Tasks

### Task 1: Even or Odd Checker (`task1.py`)
A program that determines whether a number is even or odd.

**Features:**
- Takes a number as input
- Checks if the number is divisible by 2
- Displays whether the number is even or odd

**Usage:**
```bash
python task1.py
```

**Example:**
```
Enter a number: 7
7 is an odd number.
```

### Task 2: Sum Calculator (`task2.py`)
A program that calculates the sum of all numbers from 1 to 50.

**Features:**
- Uses a for loop to iterate through numbers 1 to 50
- Calculates and displays the total sum

**Usage:**
```bash
python task2.py
```

**Output:**
```
The sum of number from 1 to 50 : 1275
```
